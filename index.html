AntiPub

A Bukkit plugin designed for anti-publicity!

Download this project as a .zip file Download this project as a tar.gz file

Welcome to a carefree world.

AntiPub is a Bukkit plugin specifically designed to block unwanted spam, including such links or IP's to advertise other servers. With flawless, streamlined code in v0.3+ it doesn't require configuration (optional) and is a light-weight plugin dedicated to helping to make your life easier.

Disclaimer:

There's no way to prevent everything (yet); however, this should to block most of any issues that it is meant to. If you are having issues, please report the issue.

Features

  • Blocks pesky individuals who choose to make your day harder (will automatically deny any submission of text that match block rules and/or settings in the configuration).
  • IPv4 addresses, http, https, ftp, and many other protocols supported by the default regex (rule).
  • IPv6 addresses are not currently supported; however, consideration will be made if deemed necessary.
  • Commands (jump to commands) will assist you in editing the configuration from the console or in-game chat (requires ).
